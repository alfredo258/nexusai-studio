# The Outbound Force — Design System

Design system extracted from the live [The Outbound Force](https://theoutboundforce.com/) website. This is the source of truth for all design decisions, colors, typography, spacing, components, and animation patterns.

## Files

| File | Description |
|------|-------------|

## Key Design Decisions

- **Light-mode violet-tinted atmosphere** — body gradient canvas, not flat white
- **Outfit + DM Sans** dual-font system with strict role separation
- **Brand-colored shadows** — `rgba(109, 40, 217, ...)` instead of neutral gray
- **Glassmorphism hero cards** — "The Pipeline" animation with morphing blobs
- **Dark process section** — Gmail + LinkedIn animated panels as background
- **SVG cat mascot** — CSS-only sleep/wake toggle near calendar CTA

## Usage

Use `DESIGN.md` as a reference for AI agents (Claude, Cursor) to generate UI that matches The Outbound Force design language. Include the relevant sections when prompting for new components or pages.
